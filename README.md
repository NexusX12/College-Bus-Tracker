## Requirements

* Node.js
* React Native
* Firebase
* React Native Paper

## Installation

1. Clone the repository
```git 
git clone https://github.com/ProjectShutdown/College-Bus-Tracker.git
```
2. Install the dependencies
```node 
npm install
```
3. Run the app
```node 
npm start
```
